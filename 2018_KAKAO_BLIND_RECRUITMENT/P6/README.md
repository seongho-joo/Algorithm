## 💡 [풀이](https://seongho96.tistory.com/31)
문제 설명에 주어진 예시를 따라 그대로 구현하면 되는 문제였다.

문제 해결 방식
1. 2차원 배열을 순회하면서 2 * 2 블록이 같은지 확인하고 따로 체크 배열을 만들어 체크한다.
2. 체크된 블록들은 `'.'`으로 바꾼다.
3. 블록들을 떨어트린다.
4. 위 과정 1 ~ 3을 사라진 블록들이 없을 때 까지 반복한다.
